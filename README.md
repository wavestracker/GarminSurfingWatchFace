# Garmin Surf watch face.

Garmin Connect IQ Watch Face, you can install through Garmin app store.

## Features
- heart rate
- calories
- do not disturb icon
- phone connection
- alarm icon
- animation for tide
- white text goes dark if device is in Sleep mode or Do Not Disturb state.
- batery
